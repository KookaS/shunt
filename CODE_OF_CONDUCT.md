# Code of Conduct

Shunt adopts the **[Contributor Covenant](https://www.contributor-covenant.org/), version
2.1**, as its code of conduct.

In short: we're committed to a harassment-free, welcoming, and respectful community for
everyone, regardless of background or identity. Be kind, assume good faith, and focus on
what's best for the community.

The full text is at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Enforcement

Report unacceptable behavior by opening a GitHub issue or contacting the maintainer directly.
Reports are handled confidentially and reviewed as promptly as a solo-maintained project
allows.
